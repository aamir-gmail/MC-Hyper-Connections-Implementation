Project Title: Manifold-Constrained Hyper-Connections (mHC) 
Transformer Description This repository contains a PyTorch implementation of Manifold-Constrained Hyper-Connections (mHC),
a novel architectural framework introduced in arXiv:2512.248801.Standard Hyper-Connections (HC) improve model
performance by expanding the residual stream width and diversifying connectivity2222.
